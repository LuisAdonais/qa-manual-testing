# Test analysis

---

## Overview

The **test analysis** phase transforms the **test basis** into **test conditions**.

Main objectives:

- Identify issues in requirements:
  - Ambiguities
  - Inconsistencies
  - Missing information
  - Risks
- Define testable conditions
- Ensure traceability between requirements and test conditions
